## ⚙️ 配置系统

### config.json 结构

```json
{
  "settings": {
    "language": "zh-CN",              // 语言设置
    "image": {
      "ratio": "5:2",                 // 图片比例
      "width": 1000,                  // 宽度
      "height": 400,                  // 高度
      "color_scheme": { ... }         // 配色方案
    },
    "content": {
      "language": "zh-CN",            // 内容语言
      "target_length": [800, 1500],   // 文章长度
      "tone": "professional_conversational"
    }
  },
  "content_templates": {
    "title_styles": [ ... ],          // 标题模板（中文）
    "hooks": [ ... ]                  // 开场白模板（中文）
  }
}
```

### 自定义选项

| 设置项 | 位置 | 说明 |
|--------|------|------|
| 图片颜色 | `color_scheme` | 3套预设配色方案 |
| 标题风格 | `title_styles` | 5种标题模板 |
| 开场白 | `hooks` | 5种开场白模板 |
| 文章长度 | `target_length` | [最小, 最大]字数 |
| 输出目录 | `output.directory` | 文件保存位置 |
