## ⚙️ 自定义配置

所有设置在 `config.json` 文件中，随时可以修改：

### 修改图片颜色

找到 `color_scheme` 部分：

**科技蓝紫风（当前）：**
```json
"primary": ["#1e3a8a", "#3b82f6", "#60a5fa"],
"accent": ["#8b5cf6", "#a78bfa"]
```

**专业绿金风：**
```json
"primary": ["#065f46", "#059669", "#10b981"],
"accent": ["#d97706", "#f59e0b"]
```

**警示红橙风：**
```json
"primary": ["#991b1b", "#dc2626", "#f87171"],
"accent": ["#ea580c", "#fb923c"]
```

### 修改标题风格

找到 `title_styles` 部分：
```json
"title_styles": [
  "🚨 {topic}：你现在必须知道的真相",
  "💡 {topic} 深度解析：从复杂到清晰",
  "📊 {topic} 完全指南：数据背后的故事"
]
```

添加你自己的模板！

### 修改文章长度

找到 `target_length`：
```json
"target_length": [800, 1500]  // 改为你想要的字数范围
```
