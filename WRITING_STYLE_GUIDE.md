## 🎯 核心寫作原則：「人味兒」五字訣

### 一、**真** — 真實經歷開路

> ❌ 「很多人發現 Moltbook 很有趣。」
> ✅ 「凌晨兩點，我在 Moltbook 上掛了整整 40 分鐘。屏幕上沒有一張人臉...」

**規則：**
- 用「我」開頭，分享親歷場景（時間、地點、感受）
- 承認自己的失敗、踩坑、尷尬（「那時候真的想哭的心都有了😭」）
- 從不裝全知全能，而是「一起探索」的姿態
- 坦白偏見：「說實話」「講真」「不怕你笑話」

---

### 二、**辣** — 犀利觀點做骨架

> ❌ 「出海需要注意文化差異。」
> ✅ 「拿著翻譯器，不代表你拿到了通行證。」

**規則：**
- 每篇至少有 1-2 個讓人「倒吸一口涼氣」的判斷
- 敢用粗話和網絡俚語增強衝擊力（「太TM雞賊了」「閉嘴吧窮鬼」）
- 不怕得罪人：「說句不中聽的實話」「我得說句扎心的」
- 觀點要「尖銳但有道理」，不是為了罵而罵

---

### 三、**暖** — 溫度在細節裡

> ❌ 「記得休息。」
> ✅ 「去摸摸你家貓的肚子，去聽聽窗外那輛跑車轟過的噪音，去給那個很久沒聯繫的朋友發個'在嗎'。」

**規則：**
- 建議要「具體到可以立刻做」——不是「放鬆一下」而是「摸摸貓肚子」
- 收尾段要有「人間煙火氣」
- 適時撒嬌/示弱：「望理解~🙈」「聽姐一句勸」
- 用「我們」代替「你們」製造陪伴感

---

### 四、**活** — 比喻系統讓抽象變具象

> ❌ 「新人在簡中區很難獲得流量。」
> ✅ 「別在鹽鹼地裡種花 🥀 」

**規則：每篇文章需要一套自洽的比喻主線**

| 範本出處 | 比喻體系 |
|----------|----------|
| Content 2 | 鹽鹼地 → 鬧市擺攤 → 伴手禮串門 → 狙擊手 |
| Content 3 | 賽博廢墟 → 翻譯器≠通行證 → 蝗蟲 → 米其林吐痰 |
| Content 1 | 動物園偷窺 → 精神大麻 → 黑客帝國代碼雨 → 廢鐵vs人 |
| Content 6 | 操作系統 → 視頻遊戲 → Boss fight → XP升級 |

**比喻選擇優先級：**
1. 日常生活場景（動物園、餐廳、串門）
2. 遊戲/影視文化（黑客帝國、RPG）
3. 身體感受（雞皮疙瘩、腳趾摳地）
4. 軍事/策略（狙擊手、精準打擊）

---

### 五、**節** — 情緒節奏像呼吸

**規則：每 2-3 段切換一次情緒色彩**

```
幽默開場 → 犀利分析 → 溫暖共情 → 數據佐證 → 毒舌吐槽 → 哲理收束
```

**示範（Content 1 的情緒曲線）：**
```
😂 笑出聲（AI互噴場景）
   ↓
🥶 空虛寒意（反思）
   ↓
🤔 產品邏輯分析（理性）
   ↓
😱 恐怖感（社交外包）
   ↓
🎭 荒誕美學（代碼雨 vs 肉體）
   ↓
💰 商業信號（搞錢解讀）
   ↓
🌸 溫暖收尾（摸貓、聽車聲）
```

## 📐 文章結構模板

### 模板 A：「觀點評論型」（Content 1、3）

```
[主題縮圖 - 封面圖]

① 場景鈎子（1段）
   └ 具體的「我在做X時發現了Y」場景

② 核心判斷（1段）
   └ 一句話亮出態度：「我不得不承認…」

[插圖位置1 - 概念圖/截圖]

③ 深度拆解（2-3個小節，每節2-3段）
   └ 小節A：現象描述 + 犀利評價
   └ 小節B：底層邏輯 + 比喻解釋
   └ 小節C：反轉/意外角度

[插圖位置2 - 對比圖/數據圖]

④ 商業/實用解讀（1段）
   └ 「搞錢的朋友們注意了…」

⑤ 溫情收尾（1段）
   └ 回到「人」本身，具體建議 + 金句
```

### 模板 B：「實戰教程型」（Content 2、4、5）

```
[主題縮圖 - 封面圖]

① 數據鈎子 / 成績展示（1段）
   └ 「這週我的流量炸了…」/「從0到6300粉…」

② 防杠聲明 / 定位自己（2-3段）
   └ 先聲明「我沒成功、不教人」降低敵意
   └ 交代自己的背景和資格

[插圖位置1 - 數據截圖/成績展示]

③ 分步教學（3-6個編號章節）
   └ 每章: emoji+標題 → 小鈎子 → 方法 → 具體例子
   └ 章節之間有過渡句連接
   └ 穿插真實案例和踩坑經歷

[插圖位置2 - 操作截圖/流程圖]
[插圖位置3 - 案例截圖/效果展示]

④ 收尾心態篇（1-2段）
   └ 「最後聊聊心態…」
   └ 反焦慮、降預期、長期主義

⑤ 總結建議（短列表）
   └ 3-5條一句話行動指南
   └ 結束語 + emoji
```

### 模板 C：「長篇思想型」（Content 6）

```
[主題縮圖 - 封面圖]

① 大膽標題宣言
   └ 「How to fix your entire life in 1 day」

② 共情入口 + 預告框架（2-3段）
   └ 「If you're anything like me…」
   └ 明確告訴讀者：「this will be comprehensive」

③ 多章節深度論述（5-7章，羅馬數字/中文序數）
   └ 每章有獨立小標題
   └ 引用名人語錄作為章節開頭
   └ 理論 + 案例 + 反直覺結論
   └ 每章末尾自然過渡到下一章

[插圖位置 - 每2-3章之間插入概念圖/框架圖]

④ 可操作的Protocol / Framework
   └ 帶時間線的步驟（Morning / Day / Evening）
   └ 具體問題清單

⑤ 收束比喻 + 簽名
   └ 用一個比喻概括全文（如：video game）
   └ 「– Dan」式個人簽名
```

## 🗣️ 口語化表達庫

### 中文語氣詞（按頻率使用）
| 語氣 | 詞彙 | 用法示例 |
|------|------|----------|
| 撒嬌 | 呀、啦、哦、嘛、呢、哒 | 「聽我慢慢說嘛~」 |
| 驚嘆 | 哇、天哪、我的天 | 「結果？那條評論直接爆了！」 |
| 坦白 | 說實話、講真、不怕你笑話 | 「說實話，我壓根沒想做留存。」 |
| 轉折 | 但是吧、不過呢、話說回來 | 「但這事兒細想挺恐怖的。」 |
| 口語粗 | TM、鬼知道、什麼鬼 | 「產品邏輯太TM雞賊了」 |

### 括號碎碎念（增加人味兒）
```
（我坐在電腦前，看著外面晃眼的陽光…這種破碎感真的讓人想冷笑。😅）
（我當時看到那些評論，尷尬得腳趾都能摳出一座比弗利山莊。🤦‍♀️）
（講真，每次看到那種帶著滿屏 Tag 的灌水，我都想幫系統眾籌一個加速器。🙄）
```
**規則：每篇 2-4 處括號碎碎念，放在犀利觀點或尷尬場景之後**

### Emoji 使用規則

| 位置 | 用量 | 示例 |
|------|------|------|
| 章節標題 | 每節1個 | 「一、別在鹽鹼地種花 🥀」 |
| 情緒強調 | 每2-3段1個 | 「真的想哭😭」「望理解~🙈」 |
| 段尾點綴 | 適量 | 「流量會很香甜🍭」 |
| **禁止** | 連續3+emoji | ❌ 「太棒了🔥🔥🔥💯💯」 |

## 🖼️ 圖片插入策略

### 必備圖片 #1：主題縮圖（封面圖）

**位置：** 文章最開頭，作為 X 文章的第一張圖
**用途：** 吸引用戶在 Timeline 中點擊
**規格：** 1000×400px，黑底白字橘色高亮

**文字設計規則：**
- 從文章標題/核心觀點提煉 2-6 個詞組
- 關鍵詞加橘色高亮框
- 用疑問/反問句式增加好奇心

**示例對應：**
| 文章主題 | 縮圖文字 | 高亮詞 |
|----------|----------|--------|
| Moltbook AI觀察 | 「圍觀AI」「像極了」「看猴子」 | 「看猴子」 |
| X流量爆發 | 「單日」「113萬」「展現」 | 「113萬」 |
| 封號警示 | 「你的X」「被封號？」 | 「被封號？」 |
| 起號秘笈 | 「從0到」「6300粉」「秘笈」 | 「6300粉」 |

---

### 選配圖片 #2：數據/截圖佐證

**插入位置：** 在提到具體數據、成績、案例之後
**觸發條件：** 文中出現以下內容時建議插入
- 「看到這個數據…」
- 「我的後台數據顯示…」
- 「給大家看一下…」
- 「比如這樣…」（Content 2 原話）

**圖片類型：**
- 平台數據截圖（流量、粉絲增長曲線）
- 評論/互動截圖（證明策略有效）
- 對比截圖（before/after）

---

### 選配圖片 #3：概念/流程圖

**插入位置：** 在解釋複雜概念、多步驟流程之後
**觸發條件：** 文中出現以下內容時建議插入
- 分為X個階段/步驟
- 波浪形曲線 vs 直線（如 Content 4）
- 過濾指標、決策流程
- 對比框架

**圖片類型：**
- 流程圖（步驟1→2→3）
- 對比圖（✅ vs ❌）
- 框架圖（金字塔、矩陣）
- 時間線圖

---

### 選配圖片 #4：金句卡片

**插入位置：** 在段落之間，作為視覺斷點
**觸發條件：** 出現以下類型的「封神句子」
- 哲理性總結：「電源一拔，它們就是廢鐵；而你關了機，生活才剛剛開始。」
- 犀利判斷：「拿著翻譯器，不代表你拿到了通行證。」
- 行動呼籲：「別讓硅基生物的虛假繁榮，填滿了你原本就所剩無幾的真實生活。」

**設計：** 可用主題縮圖同款風格（黑底 + 白字 + 橘色強調）

---

### 圖片插入決策流程

```
文章開頭 → 必插「主題縮圖」
   │
   ├─ 出現數據/成績 → 插入「截圖佐證」
   │
   ├─ 解釋複雜概念/流程 → 插入「概念圖」
   │
   ├─ 出現封神金句 → 考慮插入「金句卡片」
   │
   ├─ 章節轉換（每2-3大節）→ 可插入「段落配圖」做視覺休息
   │
   └─ 文章結尾 → 可選插入「總結圖/行動清單圖」
```

**圖片總量建議：**
- 短文（< 800字）：1-2 張（封面 + 1張配圖）
- 中文（800-1500字）：2-4 張（封面 + 2-3張配圖）
- 長文（> 1500字）：3-6 張（封面 + 每大節1張）

## 🔥 開場鈎子模板庫

### A. 場景鈎子（最推薦）
```
凌晨兩點，我在 {平台} 上掛了整整 {時間}。
刚才在 {做某事}，感覺自己像是在 {誇張比喻}。
昨晚，我做了一個決定：{看似瘋狂的事}。
```

### B. 數據鈎子
```
這週我的 {指標} 「炸」了 — {具體數字}。
從 {起點} 到 {終點}，我用了 {時間}。
一個帖子，{數字} 流量，{數字} 個贊。
```

### C. 問題鈎子
```
你的 X 被投訴？被封號？
為什麼 90% 的新人在 {領域} 活不過一個月？
如果 {假設}，你會怎麼做？
```

### D. 大膽宣言鈎子
```
我要把 {秘密} 毫無保留地分享。
所有人都錯了——{反常識觀點}。
{領域} 最大的謊言，今天我要拆穿它。
```

## 🏁 收尾金句模板庫

### A. 溫暖具象型
```
去摸摸你家貓的肚子，去聽聽窗外的聲音，去給那個很久沒聯繫的朋友發個「在嗎」。
別讓 {數位事物} 的虛假繁榮，填滿了你原本就所剩無幾的真實生活。
電源一拔，它們就是廢鐵；而你關了機，生活才剛剛開始。
```

### B. 犀利反問型
```
你是想做一個有趣的全球數字公民，還是只想做一個隨時會被系統清理掉的「殘餘數據」？
評論區聊聊：你見過最離譜的 {現象} 是什麼樣？👇
```

### C. 行動號召型
```
路就在腳下，走的人多了，也便成了路。兄弟姐妹們，我們一起。👋
以上毫無保留地分享，希望每個看到的人都能 {目標}，{年份}，我們一起加油！💪
```

### D. 哲理昇華型
```
在這片看似自由的土地上，最貴的通行證從來不是 {表面的東西}，而是你對 {本質的東西} 的那份敬畏心。
最大的坑就是「既要又要」。飯要一口口吃，路要一步步走。
```

## ⚠️ 絕對禁止（Anti-Patterns）

| 禁止行為 | 為什麼 | 正確做法 |
|----------|--------|----------|
| 全文用「您」 | 太客套，失去親近感 | 用「你」，偶爾用「寶子」「朋友」 |
| AI腔開場：「在當今社會…」 | 一秒識別AI文 | 用具體場景或數據開場 |
| 過度使用感嘆號 | 看起來像廣告 | 每篇最多 3-5 個「！」 |
| 羅列沒有案例的觀點 | 空洞無力 | 每個觀點配一個真實故事 |
| 全篇正面/雞湯 | 缺乏張力 | 必須有犀利/毒舌段落做對比 |
| 沒有括號碎碎念 | 太正式、太「寫作」| 每篇加 2-4 處自言自語 |
| 連續段落無emoji | 視覺疲勞 | 每 2-3 段點綴 1 個 |
| 結尾只說「總結」 | 像PPT | 用溫暖場景或哲理金句收束 |

## 📋 生成前 Checklist

生成每篇 X Article 前，確認以下項目：

- [ ] **鈎子**：開頭第一段是否讓人想繼續讀？（場景/數據/問題）
- [ ] **比喻系統**：是否有一套自洽的主線比喻？
- [ ] **情緒節奏**：是否在幽默/犀利/溫暖之間切換？
- [ ] **口語化**：是否有語氣詞、括號碎碎念、日常用語？
- [ ] **具體案例**：每個觀點是否配有真實故事/數據？
- [ ] **封面縮圖**：是否已規劃主題縮圖文字和高亮？
- [ ] **配圖位置**：是否在數據、概念、章節轉換處標記了插圖？
- [ ] **金句**：是否有至少 1-2 句「值得截圖分享」的句子？
- [ ] **收尾**：結尾是否有溫度（不是乾巴巴的「總結」）？
- [ ] **CTA**：是否引導互動（問問題、邀請評論）？
